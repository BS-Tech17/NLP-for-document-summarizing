# Stock Market Dashboard (Streamlit)

## Overview
This project presents an interactive stock market analytics dashboard built using Streamlit.  
It enables users to explore historical stock price movements, review financial statement data, and analyze sentiment-driven news associated with selected tickers.

The application integrates multiple financial data sources and visualizes insights through an intuitive web interface designed for academic, analytical, and demonstration purposes.

## Key Capabilities
- Interactive adjusted closing price visualization
- Daily percentage change analysis
- Distribution view of stock returns
- Annual financial statement exploration:
  - Balance Sheet
  - Income Statement
  - Cash Flow Statement
- News aggregation with sentiment indicators
- Tab-based navigation for structured analysis
- Lightweight deployment with Streamlit

## Technology Stack
- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- yFinance
- Alpha Vantage API
- StockNews
